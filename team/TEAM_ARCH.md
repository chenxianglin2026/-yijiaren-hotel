## 二、模块分工

| 模块 | 执行者 | 内容 |
|------|--------|------|
| 小程序-引流预订 | 子代理-前台 | LBS定位、房型展示、优惠券、OTA同步 |
| 小程序-入住开锁 | 子代理-前台 | 实名核验、蓝牙开锁、临时密码 |
| 小程序-在店服务 | 子代理-前台 | 客控、呼叫服务、扫码购物 |
| 小程序-退房结算 | 子代理-前台 | 一键退房、自动结算、押金秒退 |
| 小程序-周边服务 | 子代理-前台 | 餐饮推荐、洗衣寄存、本地游玩 |
| 后台-门店房态 | 子代理-后台 | 多门店、房型配置、分时定价 |
| 后台-订单会员 | 子代理-后台 | 订单管理、会员体系、营销活动 |
| 后台-财务统计 | 子代理-后台 | 营收报表、支付对账、数据分析 |
| 后台-设备监控 | 子代理-后台 | 智能设备状态、远程配置 |
| 保洁移动端 | 子代理-后台 | 工单接单、完工打卡、异常上报 |
| 后端-用户API | 子代理-后端 | 注册登录、个人信息 |
| 后端-订单API | 子代理-后端 | 预订、支付、退房 |
| 后端-设备API | 子代理-后端 | 门锁控制、客控指令 |
| 后端-对接API | 子代理-后端 | 公安旅业、OTA平台 |
| 测试 | 子代理-QA | 功能、安全、压力 |
| 审查 | OpenClaw | 代码审查、架构把关 |
